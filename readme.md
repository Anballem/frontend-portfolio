# Frontend Developer Portfolio

A responsive frontend developer portfolio built with **HTML, Tailwind CSS, and JavaScript**.

##  Features
- Responsive layout (mobile-first)
- Dark mode toggle
- Project filtering with JavaScript
- Form validation
- Smooth scrolling
- Deployed with GitHub Pages

## 🛠️ Technologies Used
- HTML5
- Tailwind CSS
- JavaScript (ES6)
- Git & GitHub
